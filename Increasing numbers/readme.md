#The problem

|   |   |    |    |    |
|---|---|----|----|----|
| 4 | 8 | 13 | 19 | 25 |
| 2 | 5 | 9  | 14 | 20 |
| 1 | 3 | 6  | 10 | 15 |

Given this table, where (x:1, y:1) corresponds to the bottom left corner,
and increasing x/ y values moves the index of the array right/ up respectively.
The table extends infinitely upwards and to the right (other directions are undefined)

Provide a function that takes a coordinate, and returns the value contained in
the respective cell of the array.
